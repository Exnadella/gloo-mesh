## meshctl cluster configure

Configure Kubernetes Cluster registered with Gloo Mesh.

### Synopsis

Create a mapping of clusters to kubeconfig entries in ${HOME}/.gloo-mesh/meshctl-config.yaml.

```
meshctl cluster configure [flags]
```
